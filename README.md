## Overview
In this lab, we learned how to create a basic REST API using **Node.js** and **Express**. We explored:
1. Creating a basic Express web app.
2. Using GET methods with and without parameters.
3. Using body-parser middleware to handle form submissions.
4. Reading JSON files with `fs`.
5. Searching JSON data.
6. Creating POST routes to handle HTML form submissions.
